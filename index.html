📚 Documentación del Proyecto

Este proyecto implementa una simulación completa del popular juego Clash Royale utilizando Java y JavaFX, aplicando conceptos avanzados de modelado y simulación de sistemas.

🎯 Objetivo del Proyecto

Desarrollar un sistema de simulación que replique la mecánica de batalla de Clash Royale, implementando algoritmos de inteligencia artificial para los personajes, gestión de recursos (elixir), y física de colisiones en tiempo real.

⚙️ Tecnologías Utilizadas

  • Java: Lenguaje de programación principal para la lógica del juego
  • JavaFX: Framework para la interfaz gráfica y animaciones
  • Programación Orientada a Objetos: Diseño modular y escalable
  • Patrones de Diseño: Factory, Strategy, Observer para gestión de entidades

🎮 Características Implementadas

  • Sistema de generación automática de elixir
  • Mecánica de despliegue de tropas y edificios
  • Inteligencia artificial para comportamiento de unidades
  • Sistema de detección de colisiones y combate
  • Animaciones fluidas y efectos visuales
  • Gestión de torres y puntos de vida
  • Sistema de victoria/derrota

🔬 Modelos de Simulación Aplicados

  • Modelo de Eventos Discretos: Para gestión de turnos y acciones
  • Sistemas Multi-Agente: Comportamiento autónomo de unidades
  • Algoritmos de Pathfinding: Navegación inteligente en el campo de batalla
  • Modelo de Recursos Limitados: Gestión de elixir y costos
